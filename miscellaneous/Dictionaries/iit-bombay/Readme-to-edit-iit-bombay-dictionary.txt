Steps to Edit/Modify iit-bombay dictionary::
============================================
1. Edit iit-bombay.txt1  present in src folder
2. dictfmt --utf8 --allchars -s "My Dictionary" -j iit-bombay < src/iit-bombay.txt1 
3. Open GoldenDict 
           ===>  Go to Edit --> Dictionaries --> Rescan now 
4. Click OK.
